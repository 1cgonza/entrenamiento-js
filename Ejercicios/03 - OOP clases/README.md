# OOP (Object Oriented Programming) y Canvas

El paradigma OOP nos permite crear objetos con diferentes propiedades. Vamos a aprender a usar clases para crear objetos en JavaScript. Los objetos son útiles en la creación de videojuegos, dibujo generativo o cualquier aplicación que requiera controlar muchos elementos en pantalla.

También vamos a comenzar a usar el Canvas de HTML5 que vamos a utilizar constantemente en las aplicaciones de ahora en adelante.

## Ejercicio

- Instalar dependencias e iniciar servidor local.
- Comentar los archivos `src/index.html`, `src/index.js` y `/src/componentes/Ojos.js`. Comentar cada linea explicando lo que hace.
- Cambiar el color de los ojos.
- Cambiar la forma de los ojos, ahora son cuadrados, cambiarlos a cualquier otra forma. (puedes intentar varias versiones para entender las posibilidades geométricas con las que podemos pintar en el Canvas)
- No dejar que los ojos se salgan del cuadro central.
- Cambiar el tamaño de los ojos dependiendo de la posición en pantalla, este cambio de tamaño debe ser gradual. Cuando el mouse se acerca a la parte baja, se hacen más pequeños y cuando se acerca a la parte alta del recuadro, se hacen más grandes.
